What it does

A complete content factory

Nine pipeline stages, one daily command. Every provider is swappable.

✦

AI ideation & scripts

Generates viral-leaning ideas and tight, retention-optimized scripts — with anti-repeat memory so topics never duplicate.

♪

Voice + word captions

Natural TTS voiceover plus word-by-word burned captions generated locally with Whisper.

▦

9:16 video assembly

ffmpeg stitches stock or AI B-roll into a 1080×1920 vertical video with motion, music ducking, and a progress bar.

⤴

Multi-platform publish

One API posts to YouTube Shorts, TikTok, Instagram & Facebook Reels — with SEO titles, tags, and AI disclosure.

⟳

Analytics feedback loop

Winning topics feed back into ideation, so the system gets sharper the longer it runs.

⚙

Web dashboard

Generate, monitor live logs, browse the library, and edit config from the browser — no terminal needed.
