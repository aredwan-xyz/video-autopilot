🚀 Get Started in 5 Steps

  1. Install once. Run bash scripts/install_mac.sh — installs ffmpeg, Python dependencies, and a caption font in one shot.
  2. Add API keys. Copy .env.example to .env and paste at minimum a Claude/OpenAI key and a Pexels key (both are free to get). The Settings → API Keys tab shows what's detected.
  3. Pick your niche. The three starter channels (Motivation, Money, Facts) are ready to go. Customise them in Settings → channels.yaml.
  4. Run a dry-run. Go to Generate, pick a channel, keep Dry-run ON, hit Start. Watch the pipeline run live and find your video in Library.
  5. Go live. Once a video looks right, configure a publish provider in settings.yaml, uncheck Dry-run, and add a cron job to run every day.
100% free to start. edge-tts (voice) + Pexels (visuals) + local Whisper (captions) = build unlimited videos for $0. Upgrade individual stages with your first revenue.

The pipeline encodes this in config/prompts/script.md. Every generated script follows this exact structure. Tune it there to match your niche voice.

🪝 Hook Formulas

The first 3 seconds decide everything. Here are the formulas that consistently stop the scroll:

  • Contrarian: "Stop doing ___ if you want ___."
  • Curiosity gap: "Nobody talks about what happens when ___."
  • Number + payoff: "5 ___ that ___ (the last one is completely free)."
  • Direct address: "If you're under 30, watch this entire video."
  • Bold claim: "This is the most underrated ___ of all time."
  • Question: "Why do the richest people in the world never ___?"
  • Warning: "Most people don't know this about ___ and it's costing them."
  • Story open: "In 2008, a broke ___ discovered something that changed everything."

The ideation prompt in config/prompts/ideation.md and script prompt in config/prompts/script.md bake these formulas into every generated video. Edit those files to customise for your niche voice.

⏱️ Retention Rules

These drive algorithmic reach more than any other factor. The pipeline enforces them automatically.

  1. Visual cut every 2–3 seconds. Static footage = viewers scrolling away. Set video.cut_every_seconds in settings.yaml.
  2. Word-by-word captions, always. ~85% of short-form video is watched muted. The pipeline burns them in at the ffmpeg stage — they're not optional overlays.
  3. No dead air. Voiceover starts at exactly 0.0 seconds. No intro music, no logo, no "hey guys."
  4. Pace audio fast. The pipeline runs TTS at 1.05–1.15× speed by default. Feels energetic, not rushed.
  5. Loop-ability. The script prompt instructs the LLM to end where it began — replays inflate watch-time metrics that the algorithm tracks.
  6. One idea per video. Confusion kills retention. If it can't be explained in a subtitle line, it's two videos.
  7. Progress bar. The pipeline optionally burns a subtle bottom-edge progress bar. Small — but it anchors viewers to completion.

  • Stagger platforms by 5 minutes. Don't fire identical posts to all four at the exact same second.
  • Batch-generate, drip-post. Generate a week of content in one session. Queue the release. Don't dump 7 videos on day 1.
  • Never post the same byte-identical file to multiple platforms. The pipeline re-encodes per platform automatically (safe zones, loudness normalisation).

🔍 SEO & Metadata

This is where you win free reach without paying for ads. The pipeline generates all of this automatically from the script — tune the metadata prompt in config/prompts/metadata.md.

  • Title: Front-load the keyword + curiosity gap. The pipeline generates 3 variants per video — pick the best one.
  • Description: 1–2 line hook + keyword + soft CTA. Keep it scannable.
  • Hashtags: 3–5 intentional tags — 1 broad (#motivation) + 3 niche-specific + 1 branded channel tag. Not just #fyp #viral.
  • Pinned first comment: A question that invites replies. Comments are a ranking signal. The pipeline suggests one per video.
  • Thumbnail (Shorts): High-contrast subject + ≤3 bold words. The pipeline outputs thumbnail text — you design the image.

📈 Scaling to a Content Factory

  1. Templatise each niche — consistent font, music mood, visual style, colour palette. Output becomes recognisable, which builds brand recall.
  2. Batch generate, drip release — run a week's worth of content in one session on Sunday, then cron posts one at a time through the week.
  3. Clone channels, not complexity — add a new block to channels.yaml, change niche + voice + audience. The daily job loops over all channels automatically.
  4. Multi-part series — break big ideas into "Part 1/5" sequences. Every part farms follows and return visits.
  5. Localise winners — swap TTS voice language + translate the script. Near-zero cost, brand new market.
  6. Kill losers fast — if a channel shows no traction after 60–90 posts and A/B testing hooks, retire it and reallocate that slot to a stronger niche.

⚖️ Compliance — Staying Un-Banned

Automation at scale is explicitly allowed on all four platforms. What gets accounts banned is spam, stolen content, and rule-breaking — none of which this system does if you follow these rules.

  1. Original > reposted. This system generates original scripts + your own AI voiceover + licensed/AI visuals. Never rip and re-upload someone else's video.
  2. Label AI content. YouTube, TikTok, and Meta all require disclosure of synthetic/AI media. Keep compliance.ai_disclosure: true in settings.yaml — it adds the flag and an "AI-assisted" line to every description automatically.
  3. No misinformation. Especially health, finance, and news. The pipeline injects "not financial advice" / "not medical advice" disclaimers for those niches automatically.
  4. No real-person impersonation or voice cloning without explicit consent from that person.
  5. Licensed assets only. Royalty-free music in assets/music/. Pexels stock is free for commercial use. Check your AI image model's commercial terms.
  6. One human-owned account per channel. 1–3 posts/day is well within every platform's limits. Don't run engagement bots or fake view services.
The one rule that kills channels: re-uploading other creators' content with a logo slapped on it. The algorithm detects it. Rights holders DMCA it. Your account gets banned. This system makes original content — use it that way.

✅ Daily Operator Checklist

Takes about 5 minutes once you're set up. Don't skip the weekly review — that's where the feedback loop closes.

Every day

  • Confirm the daily cron job ran (check logs or Library for new videos)
  • Skim each posted video for caption sync glitches or visual issues
  • Reply to 3 top comments per channel (engagement signal + content inspiration)

Every week

  • Run the analytics report: python -m src.orchestrator --report
  • Identify top 3 videos by views/saves — extract the hook formula and topic angle
  • Add winning angles to trend seeds in channels.yaml so ideation biases toward them
  • Prune any channel with zero traction after 8 weeks (kill the losers fast)
  • Refresh hook formulas in config/prompts/ideation.md and script.md
